✨ Stylized Low Poly Particle System – Unity Experiment

This project is a stylized particle system experiment developed in Unity 6.2 (6000.2.4f1), focusing on creating visually appealing, low poly effects with a cohesive artistic direction.



![PS_Rain](https://github.com/user-attachments/assets/d8f92bb9-1603-4f46-beb1-b0517ea942cb)


🎯 Overview

The goal of this project was to explore and design different types of particle effects commonly used in games, while maintaining a consistent stylized aesthetic. Each effect was crafted with both visual clarity and performance in mind.

🌧️ Included Particle Effects
Rain Particle System – Lightweight and atmospheric rainfall effect
Campfire (Fire) Effect – Stylized fire with warm tones and soft motion
Healing Effect – Subtle, magical particle system representing regeneration
Magic Swirl – A dynamic swirling effect with a mystical feel

All particle textures were handcrafted in Krita, ensuring a unique and cohesive visual identity.

🎨 Artistic Direction

The particles were primarily inspired by demonstration effects from game developers on YouTube, combined with my own artistic interpretation and stylization choices. The focus was to balance readability, style, and simplicity in a low poly context.
